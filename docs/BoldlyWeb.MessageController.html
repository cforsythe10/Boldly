    create(conn, map)

    Creates a Message.

    The attributes for the desired Message should be wrapped in a message key at the top level of the JSON load.

    Input fields are:

    content: string,
conversation_id: id,
date: date,
sent_by_creator: boolean

    Output fields can be seen in BoldlyWeb.MessageView.render/2.
